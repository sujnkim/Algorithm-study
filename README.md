## 진행방식
* 매 주 2회(목요일, 일요일) 온라인 모임
  * 목 : 오전 10:00, 책 PART2. 기본적인 개념 정리 및 책 문제
  * 일 : 오후 21:00, 책 PART3. 문제 풀이(백준, 카카오 등)
* 모임에서 해야하는 활동
  * 이론 정리 공부 및 발표
  * 공부한 개념 점검
  * 어려웠던 문제 확인, 상호 피드백
